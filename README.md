# Introduction to Machine Learning with Python: Summary Notes

**Original Authors:** Andreas C. Mueller & Sarah Guido

This repository contains a collection of Jupyter Notebooks summarizing the core concepts from the book **"Introduction to Machine Learning with Python: A Guide for Data Scientists"**. These notes are structured to facilitate the understanding of fundamental Machine Learning principles, algorithm implementation using Python (specifically `scikit-learn`), and best practices in data processing and model evaluation.

## 📂 Repository Structure

The summaries are divided into 8 main modules that follow the learning path of the book:

* **`01_Introduction.ipynb`**
    A basic introduction to Machine Learning, essential libraries (`scikit-learn`, `NumPy`, `pandas`, `matplotlib`), and the construction of a first classification model.
* **`02_supervised_learning.ipynb`**
    Covers various Supervised Learning algorithms for both classification and regression tasks, including k-NN, Linear Models, Decision Trees, Random Forests, SVMs, and Neural Networks.
* **`03_unsupervised_learning.ipynb`**
    An exploration of Unsupervised Learning techniques, focusing on dimensionality reduction (PCA, t-SNE) and clustering algorithms (k-Means, DBSCAN).
* **`04_representing_data_and_engineering_features.ipynb`**
    Techniques for representing data to optimize Machine Learning algorithms, covering topics like one-hot encoding, binning, and non-linear feature transformations.
* **`05_model_evaluation_and_improvement.ipynb`**
    Advanced methods for evaluating model performance, including Cross-Validation, Grid Search for hyperparameter tuning, and an overview of various evaluation metrics.
* **`06_algorithm_chains_and_pipelines.ipynb`**
    Utilizing the `Pipeline` class in `scikit-learn` to chain multiple processing steps (such as data scaling and model training) to streamline workflows and prevent data leakage.
* **`07_working_with_text_data.ipynb`**
    The application of Machine Learning to Natural Language Processing (NLP), including text data representation (Bag of Words) and feature extraction techniques for documents.
* **`08_wrapping_up.ipynb`**
    Conclusion, a summary of the standard data science workflow, and recommendations for further advancement in the field of Machine Learning.

## 🛠️ System Requirements

To run the `.ipynb` files in this repository, it is recommended to set up a Python environment with the following libraries:
* `scikit-learn`
* `numpy`
* `pandas`
* `matplotlib`
* `scipy`
* `mglearn` (a specific helper library provided by the book)
